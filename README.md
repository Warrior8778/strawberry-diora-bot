# 🍓 Strawberry_Diora — Telegram Bot

Универсальный бот-администратор для компании Strawberry_Diora.

## 🚀 Возможности

### Для клиентов
- 📦 **Оформление заказов** — пошаговое заполнение с доставкой
- 📅 **Бронирование стола** — выбор даты, времени, количества гостей
- 🔍 **История заказов** — статус в реальном времени
- 💬 **AI-чат** — умный ассистент на базе Claude отвечает на любые вопросы
- ❓ **FAQ** — частые вопросы о доставке и меню
- 📞 **Контакты** — адрес, телефон, сайт

### Для администраторов
- 📋 **Управление заказами** — принять / отправить / выполнить / отменить
- 📅 **Бронирования** — подтвердить или отклонить
- ✅ **Задачи команды** — создание и выполнение задач с приоритетами
- 🔔 **Уведомления** — мгновенные алерты о новых заказах и бронях

## ⚙️ Установка

### 1. Клонировать / скачать проект
```bash
cd strawberry_diora_bot
```

### 2. Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Настроить переменные окружения
```bash
cp .env.example .env
```

Отредактируйте `.env`:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
ANTHROPIC_API_KEY=ваш_ключ_anthropic
ADMIN_IDS=ваш_telegram_id  # узнать у @userinfobot
```

### 4. Запустить бота
```bash
python bot.py
```

## 📱 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/admin` | Панель администратора |
| `/newtask` | Создать задачу (только для admin) |
| `/reset` | Очистить историю AI-диалога |
| `/cancel` | Отменить текущее действие |

## 📊 Статусы заказов

| Статус | Описание |
|--------|----------|
| 🆕 `new` | Новый заказ |
| ✅ `accepted` | Принят в работу |
| 🚀 `delivery` | В доставке |
| ✔️ `done` | Выполнен |
| ❌ `cancelled` | Отменён |

## 🛠 Технологии

- **Python 3.10+**
- **python-telegram-bot 21** — Telegram Bot API
- **Anthropic Claude** — AI-ассистент
- **SQLite + aiosqlite** — хранение данных
- **python-dotenv** — конфигурация

## 💡 Как узнать свой Telegram ID?
Напишите боту [@userinfobot](https://t.me/userinfobot) — он пришлёт ваш ID.

## 🤖 Как создать Telegram-бота?
1. Напишите [@BotFather](https://t.me/BotFather) команду `/newbot`
2. Придумайте имя и username
3. Скопируйте токен в `.env`
